## How GitHub Copilot Cloud mode works (short version)

When you start Copilot in **Cloud mode** from VS Code, your coding task runs on a remote cloud environment instead of your laptop.  
Copilot reads your repository, makes changes there, and reports progress back to GitHub/VS Code.

## Can you control it from mobile after starting on laptop?

**Partly, yes.**  
From a mobile browser, you can usually monitor progress and leave comments/instructions through GitHub (for example on the related PR or task thread).

**But full IDE-style control is no.**  
Deep editing, terminal use, and rich code navigation are still best done from VS Code on a computer.
